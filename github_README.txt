Welcome to the Github repository for SPECL, the Systems Paleoecology and Ecosystems Complexity Laboratory, a section of the Roopnarine Lab at the California Academy of Sciences. This repository hosts, or will host, code used for various projects developed and contributed by various lab members and collaborators, e.g. code for the CEG (Cascading Extinction on Graphs) family of models.

The repository is maintained by Peter Roopnarine, and help can always be requested via email to proopnarine@calacademy.org
